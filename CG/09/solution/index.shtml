Exercise 1: Ray-Box Intersection

The AABB intersection uses the slab method. Each axis produces an interval of valid ray parameters. The ray hits the box only if the three axis intervals overlap. The input values tMin and tMax let the traversal limit the box test to the closest hit found so far. The BVH uses the boolean wrapper intersect, while intersectInterval computes the actual entry and exit distances, just as in the octree exercise.

Exercise 2: BVH Construction

The builder leaves the scene objects in their original vector and only reorders bvhObjectIndices. Each node stores the bounding box of its object range. Small ranges become leaves. Larger ranges are split at the median of the centers along the axis with the largest center extent.

Exercise 3: BVH Traversal

Traversal first tests the node's box. If that box is missed, the entire subtree can be skipped. Leaf nodes test their small primitive range directly. Inner nodes traverse both children and shrink maxT as soon as a nearer intersection is found.
