Exercise: HSV to RGB conversion (15 points)

Familiarize yourself with the methods init and mouseMove. The overridden init method iterates over an image of a given width and height.

The image pixels are colored with convertPosFromHSVToRGB. image.setNormalizedValue expects RGB values in the range [0, 1]. At the moment, the returned color uses the x-coordinate as red, the y-coordinate as green, and 1 as blue. The image below shows this initial output.

Initial rendering.

While the program is running, mouseMove prints a debug message that shows both the HSV and RGB value of the pixel under the mouse cursor. The debug output uses convertPosFromHSVToRGB to compute the RGB value from the expected HSV color. The horizontal mouse position should correspond to hue (H) multiplied by 360, and the vertical position should correspond to saturation (S). This is not implemented correctly yet.
