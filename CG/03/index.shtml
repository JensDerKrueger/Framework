Sheet 3: Splines

Exercise: Spline Drawing (10 points)

Complete the function drawPolySegment. The function receives the geometry matrix of the curve method and the curve parameters, then draws the resulting curve as a polyline.

The curve methods in this program are Hermite, Bezier, and B-spline curves. Each curve is described by five parameters: four control values and one color. For a Hermite curve, these values are two points, two tangent vectors, and a color.

Write the result to the vector curve. Each point along the polyline must use this format: 

[x, y, z, r, g, b, a]

With a correct implementation, the program draws three curves: a Hermite curve at the top, a Bezier curve in the middle, and a B-spline at the bottom.

Loading

Rendering of the three spline segments: Hermite at the top, Bezier in the middle, and B-spline at the bottom. The Bezier curve is drawn twice; the second version uses the De Casteljau algorithm.
