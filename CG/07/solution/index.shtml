Exercise 1: Primary Rays and Supersampling

The renderer iterates over all pixels of the target image and creates a primary ray through the camera for each pixel. With a single sample, the pixel position is used directly. With multiple samples per pixel, samples are distributed uniformly inside the pixel, each ray is traced separately, and the resulting colors are averaged.

The averaged color is then written to the three color channels, and alpha is set to fully visible. This makes Raytracer::render fill the given image exactly where the starter code still had the TODO marker.

Exercise 2: Sphere Intersection

The sphere is intersected with the geometric ray-sphere test. First, the vector from the ray origin to the sphere center is projected onto the ray direction. If the center lies behind the ray, or if the distance from the ray line to the center is larger than the radius, there is no hit.

Otherwise, the distance along the ray direction gives the front intersection. If the ray starts inside the sphere, the back intersection is used instead. The normal is the normalized vector from the sphere center to the hit point.

Scene Intersection and Background

Scene::traceLocalRay first searches for the nearest intersection of the ray with the scene. If the ray hits no object, the background color is returned. The function also contains a simple termination condition based on recursion depth; for sheet 7, local lighting is the important part, but the signature is already prepared for the following ray tracing exercises.

Shadows and Local Lighting

For each point light, a shadow ray is created from the hit point toward the light source. The shadow ray origin is shifted by a small OFFSET_EPSILON along the normal so rounding errors do not make the ray immediately hit the same surface again.

If no blocking intersection lies between the surface and the light, the solution adds ambient, diffuse, and specular Phong terms. If the point is in shadow, only the ambient term remains. Because the loop runs over all lights, shadows naturally support multiple point lights.
