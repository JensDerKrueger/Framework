Exercise: Barycentric Interpolation (10 points)

Barycentric coordinates can describe the position of a point relative to a triangle. The three vertices V0, V1, and V2 form a triangle in two-dimensional space. Their positions and RGB colors are: 

P₀ = (−3, 4)  C₀ = (1.0, 0.15, 0.15)
P₁ = (5, 2)   C₁ = (0.15, 1.0, 0.15)
P₂ = (1, 0)   C₂ = (0.15, 0.15, 1.0)

Compute the barycentric coordinates α0, α1, and α2 of point Q and explain whether Q lies inside triangle P0P1P2. If it does, also give the interpolated color. Round your results to two decimal places. 

a) Q = (0, 0)   b) Q = (2, 1)

Exercise: Ray-Sphere Intersection (10 points)

Download the starter code for this sheet from Moodle. The code in init in main.cpp defines a sphere in three-dimensional space by its radius and center.

The program iterates over a virtual screen grid and colors each pixel according to the intersection test. Implement the ray-sphere intersection in raySphereIntersect. Use rayOrigin and pixelPos to define a ray and test that ray against the sphere.

If the ray hits the sphere, return the intersection point. With a correct implementation, a shiny red sphere lit from the upper front should appear.

Lade

Hold down the left mouse button to rotate the light source around the sphere. Press R to reset the light rotation.
