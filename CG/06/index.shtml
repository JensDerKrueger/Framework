Exercise: Procedural bump mapping (10 points)

In this exercise you implement bump mapping, a technique that modifies the surface normal so the surface appears uneven. This makes it possible to suggest complex surfaces with very few primitives.

Implement BumpPhongShader and apply the function f(x,y) to the surface position. Since f(x,y) is continuous, its partial derivatives with respect to x and y can be used to compute a new normal at each position. 

z  = f(x, y)   = sin²(πx) ⋅ sin²(πy)
∂z/∂x = fx(x,y) = 2π⋅sin²(πy)⋅cos(πx)⋅sin(πx)
∂z/∂y = fy(x,y) = 2π⋅sin²(πx)⋅cos(πy)⋅sin(πy)

In this simplified version of the method, the original interpolated normals do not need to be adjusted. Instead, replace them with the vector (-∂z/∂x, -∂z/∂y, 1).

BumpPhongShader receives two parameters: cellSize and bumpHeight. cellSize is especially important because it defines the x-y scaling of f(x,y).
