Store Vertex Attributes

To make it possible to interpolate all attributes across a triangle, a Vertex stores not only its position but also material data and a normal. The constructors set useful defaults so the simple test cases in main.cpp can stay concise.

Triangle and Helper Functions

Triangle stores its three vertices and a reference to the shader being used. It also provides helper functions for linear interpolation and signed area computation. That area is later used to compute barycentric coordinates.

Exercise 1: Rasterization and Interpolation

Triangle::draw first computes the total area of the triangle and then a simple bounding box in pixel coordinates. For each pixel in that box, the three barycentric weights are computed from subareas. If all weights are in the valid range, the pixel lies inside the triangle.

For valid pixels, a new surface vertex is assembled. Position, normal, and material colors are interpolated with the same barycentric weights. After that, only the associated shader decides which color is written to the image.

Area and Attribute Blending

Interpolation is identical for all Vec3 attributes: a0 * val0 + a1 * val1 + a2 * val2. The area function uses only the x- and y-coordinates of the vertices because the triangles in this exercise lie in the image plane.

Ambient Shader

The ambient shader is a pass-through shader: it directly returns the interpolated ambient material color of the current surface point. This makes the first triangle show that vertex colors are blended correctly across the surface.

Exercise 2: Diffuse Lighting

DiffuseShader stores the position of the point light and its diffuse light color. For every rasterized surface point, the normal and light direction are normalized. The diffuse factor is the clamped dot product max(0, dot(N, L)).

The final color is formed from the material diffuse color, the light diffuse color, and this diffuse factor. Because the shader works per pixel on the already interpolated surface vertex, it produces the per-pixel diffuse lighting requested by the assignment.
