This page contains the exercise sheets for the lecture "Introduction to Computer Graphics". Alongside the assignments, you can run the reference solutions directly in your browser.

The solutions were built from the original C++ exercise programs and compiled to WebAssembly with Emscripten. WebAssembly (WebAssembly.org) makes it possible to run high-performance code directly in the browser, so you can explore the computer graphics examples interactively on any operating system.
