Exercise: Marching Cubes (15 points)

Do not change any helper classes in the provided Utils folder!

Before starting the assignment, make sure you are familiar with the files Volume.h and MC.inl. If you are using Visual Studio, move the files bonsai.dat and bonsai.raw from the 05_MC folder to the VS subfolder.

In this assignment, you will implement the marching cubes algorithm as discussed in the lecture. Your task is to complete the constructor of the Isosurface class in MC.cpp. The result should resemble the figure above.

You can change the isovalue interactively by pressing Up and Down.

The algorithm works by traversing all voxels in the volume data. For each voxel, retrieve the scalar values and normals at the corners. Compare these values against the selected isovalue to determine the case configuration.

Once the case is known, interpolate the vertex positions and normals along the voxel edges, and generate the corresponding triangles to add to the isosurface mesh.
