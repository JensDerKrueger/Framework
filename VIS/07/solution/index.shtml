Setup

The solution keeps a copy of the original noise image, stores the current LIC result in licImage, and uses steps = 50 as the default kernel length in both streamline directions. Initialization computes the first LIC result from the noise image.

Streamline Tracing

LIC needs the streamline through each output pixel. The solution fixes the three-dimensional flow field at z = 0.5, converts the vector to its x/y direction, and traces forward and backward with an Euler step. Tracing stops when the curve leaves the unit square.

LIC Averaging

For every output pixel, licStep computes the streamline through the pixel and samples the input noise along that curve. The average sampled value becomes the output intensity and is written to all three RGB channels.

Histogram Equalization

After each LIC pass, the result is contrast-enhanced with histogram equalization. The solution builds a histogram, converts it to a cumulative distribution, creates a remapping table, and applies the remapped grayscale value to all color channels.

Repeated LIC Passes

The assignment asks for LIC followed by histogram equalization, repeated three times. computeLIC implements exactly that loop, feeding each equalized result back as the input for the next pass.
