Exercise: Grayscale (5 points)

Do not change any helper classes in the provided Utils folder!

Before starting with the assignment, make yourself familiar with the Image and Grid2D classes provided in the Utils folder.

Your first task is to implement the function toGrayscale in main.cpp. The result should be similar to the interactive code below. Your implementation should include two grayscale modes: uniform and non-uniform.

  • Uniform: each RGB component is equally weighted.
  • Non-uniform: R, G, and B are weighted with 0.299, 0.587, and 0.114 respectively.

You can test your output by pressing G for non-uniform and H for uniform grayscale. Press R to reset the image.

Exercise: Filter (5 points)

For this task, you must implement the function filter. The input is a 2D grid (kernel) representing the filter to apply.

Apply this kernel to the image as explained in the lecture. You can test your results with different filters using keyboard shortcuts:

  • M – Mean filter
  • A / B – Sobel filters in different directions

Exercise: ASCII Art (5 points)

Your final task is to implement the function toString. This function should convert the image into a string representing an ASCII art version and write it to ascii.txt.

To reduce the output size, use only every fourth pixel from the original image.

You can compare your results to the reference file provided here. For a better viewing experience, consider adjusting the resolution of your text editor. The file is generated by pressing T.

Try pressing: M Mean Filter, A horizontal Sobel, B vertical Sobel,
G Weighted Grayscale, H Uniform Grayscale, R Reset
