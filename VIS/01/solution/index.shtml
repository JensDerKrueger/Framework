Simulation Parameters

The implementation stores the previous, current, and next height fields in three separate grids. The constants alpha and beta are derived from the wave speed, grid spacing, and timestep and are reused for every grid cell.

Water Height Update

For each grid position the solution reads the four direct neighbors from the current height field, the current center value, and the matching value from the previous height field. The modulo operations wrap the indices at the image boundaries.

The missing line is the explicit update v = alpha * (left + right + top + bottom) + beta * center - past. Positive heights are drawn in the red channel, negative heights in the green channel.

Advancing the Time Step

After all cells have been computed, the three grids are rotated. The old current grid becomes the previous grid, the newly computed grid becomes current, and the old previous grid is reused as the next output buffer in the following frame.

Mouse Interaction

A mouse click writes an alternating positive or negative impulse into the current height field. The following animation frames propagate that impulse through the finite-difference update above.
