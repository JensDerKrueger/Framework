Task Overview

The exercise compares streamlines, pathlines, and streaklines in a time-dependent flow field. The solution precomputes one line buffer for each line type and lets the user cycle between them with the space bar.

Streamlines

A streamline follows the vector field at one fixed time. The solution generates random seed points and advects each line with t = 0 for every step, so the curve shows the instantaneous direction of the flow at that time slice. 

Loading main.cpp...

Pathlines

A pathline follows one particle through the time-dependent vector field. The solution starts from a random seed, advects from the previous point, and advances the time variable by deltaT after each successful step.

Streaklines

A streakline shows the positions of particles released from the same seed point at different times. For every output sample, the solution restarts at the seed and integrates forward for a different number of time steps before storing the point.

Shared Euler Step

All three line types use the same single-step advection helper. It stops integration once a point leaves the unit cube; otherwise it performs an explicit Euler step using flow.interpolate(position, t).
