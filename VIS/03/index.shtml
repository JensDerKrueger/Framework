Do not change any helper classes in the provided Utils folder!

In this assignment, you will implement the marching squares algorithm as discussed in the lecture. Implement the algorithm inside the constructor of the Isoline class in MS.cpp. The result of your implementation should be similar to the figure above.

The marching squares algorithm extracts isolines corresponding to a given isovalue from 2D scalar fields. The algorithm iterates over all cells in the image domain and retrieves the scalar values at the corners of each cell. These values are then compared to the isovalue to determine the specific case for the cell.

Based on the case, the algorithm interpolates along the cell edges to calculate the start and endpoints of the isolines. In ambiguous cases, you must use the asymptotic decider to determine the actual topology of the isoline within the cell.
