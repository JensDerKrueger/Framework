On this website, you’ll find the exercises for the lecture "Computer Graphics Master". Alongside the tasks, you can also see the solutions running live in your browser. Our platform allows you to run the programs directly in your browser – no additional installation or setup required.

The solutions were compiled from the original C++ exercise programs and translated into WebAssembly using the Emscripten framework. WebAssembly (WebAssembly.org) is a modern technology that allows high-performance code to run directly in the browser. This enables you to experience practical computer graphics in real time, regardless of your operating system.
