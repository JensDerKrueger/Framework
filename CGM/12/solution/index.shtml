Regular Sample Positions

A regular N x N pattern divides each pixel into equal sub-cells. The sample is placed at the center of each sub-cell. This increases the sampling frequency by evaluating the continuous signal multiple times per pixel.

Jittered Stratified Samples

Jittering removes the globally repeated sample locations of the regular grid. The important detail is that the random offset is local to the current stratum: every stratum still contains exactly one sample, so the sampling pattern remains well distributed inside the pixel.

Sample Counts

The sample counts are chosen so the four images improve in reading order. Regular supersampling uses 4 x 4 samples. The jittered image uses 8 x 8 samples, which keeps the noise lower than a low-sample jittered image. The reference image uses 16 x 16 jittered samples and should therefore be the smoothest.

Pixel Integration

The sampling loop approximates the color of a pixel by averaging several evaluations of the continuous image signal. The point sample uses only the pixel center. All other modes average an N x N grid of sub-samples.
