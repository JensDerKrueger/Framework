G-buffer Setup

The first pass renders the scene into four floating point textures and a depth renderbuffer. The color clear uses alpha 0, while covered fragments write position alpha 1. This gives the deferred shader a simple way to distinguish geometry pixels from the background.

G-buffer Shader

The G-buffer fragment shader stores material and geometry information instead of computing lighting. Normal mapping is still performed in this pass, because the final view-space normal is exactly the quantity the deferred shader needs later.

Deferred Lighting And Debug View

The second pass binds the G-buffer textures and draws one full-screen triangle. The fragment shader reconstructs the surface data per pixel and computes Phong lighting in image space. The debug path uses the same textures but remaps the texture coordinates into four screen quadrants.
