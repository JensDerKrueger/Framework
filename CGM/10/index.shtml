Sheet 10: Deferred Shading

Exercise: Deferred Shading (25 points)

In the previous OpenGL exercises, the fragment shader immediately computed the final color of each visible surface point. Deferred shading separates this into two stages. First, the scene is rendered into several textures, the G-buffer. Then a second full-screen pass reads those textures and computes the lighting in image space.

The provided project already contains the scene, camera controls, texture loading, vertex buffers, and a full-screen triangle. Your task is to complete the deferred rendering pipeline in main.cpp, Shader/gbuffer.frag, and Shader/deferred.frag.

The G-buffer stores four pieces of information for every visible pixel: diffuse color, specular color, view-space normal, and view-space position. Press D to toggle between the final shaded image and a 2 x 2 debug view of these intermediate buffers. Press Space to animate the light, R to reset the view, and drag with the left mouse button to rotate the scene.

Task 1: Create And Fill The G-buffer (8 points)

Complete ensureGBufferSize and the first pass in draw. The four textures diffuseBuffer, specularBuffer, normalBuffer, and positionBuffer should have the current framebuffer size and four floating point components. The depth buffer is needed so the first pass still performs normal visibility testing.

  1. Return early if the G-buffer already has the correct size.
  2. Allocate the four textures with GLTexture2D::setEmpty.
  3. Resize depthBuffer with setSize.
  4. Bind the four color buffers and depth buffer with gBuffer.bind.
  5. Clear the G-buffer and call renderSceneToGBuffer.
  6. Unbind the framebuffer before rendering the final image.

Task 2: Write The G-buffer Shader (7 points)

Complete Shader/gbuffer.frag. This shader does not compute lighting. It only records the surface data needed by the later lighting pass.

The diffuse and specular outputs should use the material colors and, when enabled, multiply them with the corresponding textures. The normal output should store a normalized view-space normal. If normal mapping is enabled, transform the normal-map vector from tangent space to view space using the tangent, binormal, and normal vectors. Since texture values are in [0, 1], encode the final normal from [-1, 1] back to [0, 1] before storing it.

Task 3: Deferred Lighting (7 points)

Complete renderDeferredImage in main.cpp and shadePixel in Shader/deferred.frag. The deferred pass renders one full-screen triangle. For each screen pixel, the fragment shader reads the four G-buffer textures and computes Phong illumination in view space.

The position buffer alpha marks whether the pixel belongs to geometry. Background pixels have alpha 0 and should stay black. For visible pixels, reconstruct the normal from the encoded normal buffer, compute the light vector from the stored position to lightPosition, and use the camera at the view-space origin for the view vector.

Task 4: Buffer Debug View (3 points)

Complete showBuffer in Shader/deferred.frag. Split the screen into four quadrants and display the four G-buffer textures. A useful layout is diffuse in the top-left, specular in the top-right, normal in the bottom-left, and position in the bottom-right. The position values are view-space coordinates, so remap them into a visible color range before displaying them.
