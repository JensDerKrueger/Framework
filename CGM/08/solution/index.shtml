Volume Generation

The generator first creates a volume with the requested dimensions and maps every voxel to normalized coordinates in [0, 1]. For each voxel, it computes fractal noise, a soft cloud envelope, high-frequency detail erosion, and then stores the final density as an 8-bit value.

Fractal Noise

The rescale-and-add part evaluates several octaves of Perlin noise. After each octave, roughness reduces the amplitude and lacunarity increases the frequency. The accumulated value is divided by the sum of amplitudes so that changing the octave count does not change the overall value range too much.

Perlin Noise

The Perlin implementation finds the lattice cell containing the sample point, computes smooth interpolation weights with fade, evaluates hashed gradient vectors at the cell corners, and trilinearly interpolates those contributions.

Cloud Envelope

The envelope keeps the volume brick rectangular but fades the actual cloud density near a soft spherical boundary. A low-frequency Perlin sample perturbs the radius, which makes the silhouette less perfectly round.
