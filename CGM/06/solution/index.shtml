Front-To-Back Compositing

The helper function under adds one newly sampled color to the accumulated result. The new sample contributes only through the opacity that is still missing in the current result. This is why the factor 1.0 - last.a appears in both the RGB and alpha update.

Ray Setup

The shader first computes how far the ray travels through the clipped volume. The sample count is estimated from the number of voxels crossed by the ray and scaled by the interactive oversampling value. The step length and delta vector then describe one movement from sample to sample.

Ray-Marching Loop

Each iteration samples the scalar volume texture, maps the scalar through the transfer function, corrects the opacity for the current step length, and composites the sample into the accumulated color. Once the accumulated opacity is high enough, the remaining samples would barely change the result, so the loop can stop early.
