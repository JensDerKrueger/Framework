Recursion and Ray Setup

Recursive ray tracing needs a termination condition before any secondary rays are spawned. If the recursion depth reaches zero, the ray returns the background color. Otherwise the ray is intersected with the scene. A miss also returns the background color.

Reflection and Refraction

Reflective materials spawn a reflected secondary ray. The new ray origin is shifted slightly along the surface normal so the ray does not immediately hit the same surface again due to numerical precision. 

Loading Scene.cpp...

Refractive materials try to compute a transmitted direction with Vec3::refract. If refraction succeeds, the sign of the dot product between refracted direction and normal tells whether the ray leaves the material or enters it. The recursive call then uses the corresponding index of refraction.

Local Illumination and Composition

The local Phong illumination from the starter code stays mostly unchanged. It still computes ambient, diffuse, and specular light, and still uses shadow rays to decide whether direct light reaches the hit point.

The final color is a weighted sum of local illumination, reflected color, and refracted color. Purely local materials use only the local term. Reflective materials use Schlick reflectivity for the reflection weight. Refractive materials use the material's local fraction first and split the remaining contribution between reflection and transmission.

Schlick Reflectivity

Material::getReflectivity implements Schlick's approximation. For refractive materials, R0 is computed from the index of refraction. For reflective but non-refractive materials, the simplified exercise model uses 1 - local as the base reflectivity.
