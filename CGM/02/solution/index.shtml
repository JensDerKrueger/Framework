Task 1: Ray-Box Intersection

The AABB intersection uses the slab method. Each axis produces one interval of valid ray parameters. The ray hits the box only if the three axis intervals overlap. The input values tMin and tMax allow traversal to clip the box test to the nearest hit found so far. The BVH uses the boolean wrapper intersect, while intersectInterval computes the actual entry and exit distances just like in the octree exercise.

Task 2: BVH Construction

The builder keeps the scene objects in their original vector and only reorders bvhObjectIndices. Each node stores the bounding box of its object range. Small ranges become leaves. Larger ranges are split at the median centroid on the axis with the largest centroid extent.

Task 3: BVH Traversal

Traversal first checks the node box. A missed box rejects the whole subtree. Leaf nodes test their small primitive range directly. Inner nodes recurse into both children and tighten maxT whenever a closer intersection is found.
